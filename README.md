# 이노문제답안

이 저장소는 이노베이션 아카데미 `문제.txt` 중 10개 문제를 비개발자도 이해할 수 있도록 프로그램 설계 형태로 정리한 문서 저장소입니다.

## 문서 구성

- [10개 문제 프로그램 설계](./10개_문제_프로그램_설계.md)

## 읽는 방법

각 문제는 아래 순서로 정리했습니다.

1. 무엇을 만드는가
2. 왜 필요한가
3. 쉬운 비유
4. 필요한 부품
5. 동작 순서
6. 화면 또는 결과물
7. 확인 방법
8. 조심할 점

개발 경험이 없는 사람도 설명할 수 있도록 어려운 용어는 최대한 풀어서 작성했습니다.
